　 何 でも 大きな 船 に 乗る て いる 。

「 この 船 は 西 へ 行く ん です か 」

「 なぜ 」 と 問い返す た 。
「 落ちる て 行く 日 を 追 かける よう だ から 」

「 西 へ 行く 日 の 、 果 は 東 か 。 それ は 本 真 か 。 東 出る 日 の 、 御 里 は 西 か 。 それ も 本 真 か 。 身 は 波 の 上 。 枕 。 流せる 流す 」 と 囃す て いる 。 舳 へ 行く て 見る た 、 水夫 が 大勢 寄る て 、 太い 帆綱 を 手繰る て いる た 。
　 自分 は 大変 心細い なる た 。 いる つ 陸 へ 上がれる 事 か 分る ない 。 そう する て どこ へ 行く の だ か 知れる ない 。 ただ 黒い 煙 を 吐く て 波 を 切る て 行く 事 だけ は たしか だ ある 。 その 波 は すこぶる 広い もの だ ある た 。 際限 も ない 蒼い 見える 。 時には 紫 に も なる た 。 ただ 船 の 動く 周囲 だけ は いつ でも 真白 に 泡 を 吹く て いる た 。 自分 は 大変 心細い た 。 こんな 船 に いる より いっそ 身 を 投げる て 死ぬ で しまう う か と 思う た 。
　 乗合 は たくさん いる た 。 たいてい は 異人 の よう だ ある た 。 しかし いろいろ だ 顔 を する て いる た 。 空 が 曇る て 船 が 揺れる た 時 、 一 人 の 女 が 欄 に 倚 り かかる て 、 しきりに 泣く て いる た 。 眼 を 拭く 手巾 の 色 が 白い 見える た 。 しかし 身体 に は 更紗 の よう だ 洋服 を 着る て いる た 。 この 女 を 見る た 時 に 、 悲しい の は 自分 ばかり だ は ない の だ と 気がつく た 。
　 ある 晩 ｜ 甲板 の 上 に 出る て 、 一 人 で 星 を 眺める て いる た 、 一 人 の 異人 が 来る て 、 天文学 を 知る てる か と 尋ねる た 。 自分 は つまらない から 死ぬ う と さえ 思う て いる 。 天文学 など を 知る 必要 が ない 。 黙る て いる た 。 すると その 異人 が 金 牛 宮 の 頂 に ある 七 星 の 話 を する て 聞く せる た 。 そうして 星 も 海 も みんな 神 の 作る た もの だ と 云う た 。 最後 に 自分 に 神 を 信仰 する か と 尋ねる た 。 自分 は 空 を 見る て 黙る て いる た 。
　 或 時 サローン に 這入る た 派手 だ 衣裳 を 着る た 若い 女 が 向う むき に なる て 、 洋琴 を 弾く て いる た 。 その 傍 に 背 の 高い 立派 だ 男 が 立つ て 、 唱歌 を 唄う て いる 。 その 口 が 大変 大きい 見える た 。 けれども 二 人 は 二 人 以外 の 事 に は まるで 頓着 する て いる ない 様子 だ ある た 。 船 に 乗る て いる 事 さえ 忘れる て いる よう だ ある た 。

　 家 に は 若い 母 と 三つ に なる 子供 が いる 。 父 は どこ か へ 行く た 。 父 が どこ か へ 行く た の は 、 月 の 出る て いる ない 夜中 だ ある た 。 床 の 上 で 草鞋 を 穿く て 、 黒い 頭巾 を 被る て 、 勝手口 から 出る て 行く た 。 その 時 母 の 持つ て いる た 雪洞 の 灯 が 暗い 闇 に 細長い 射す て 、 生垣 の 手前 に ある 古い 檜 を 照らす た 。
　 父 は それ きり 帰る て 来る ない た 。 母 は 毎日 三つ に なる 子供 に 「 御 父様 は 」 と 聞く て いる 。 子供 は 何 と も 云う ない た 。 しばらく する て から 「 あっち 」 と 答える よう に なる た 。 母 が 「 いる つ 御 帰り 」 と 聞く て も やはり 「 あっち 」 と 答える て 笑う て いる た 。 その 時 は 母 も 笑う た 。 そうして 「 今に 御 帰り 」 と 云う 言葉 を 何 遍 と ない 繰返す て 教える た 。 けれども 子供 は 「 今に 」 だけ を 覚える た のみ だ ある 。 時々 は 「 御 父様 は どこ 」 と 聞く れる て 「 今に 」 と 答える 事 も ある た 。
　 夜 に なる て 、 四隣 が 静まる と 、 母 は 帯 を 締める 直す て 、 鮫 鞘 の 短刀 を 帯 の 間 へ 差す て 、 子供 を 細 帯 で 背中 へ 背負う て 、 そっと 潜り から 出る て 行く 。 母 は いつ でも 草履 を 穿く て いる た 。 子供 は この 草履 の 音 を 聞く ながら 母 の 背中 で 寝る て しまう 事 も ある た 。
　 土塀 の 続く て いる 屋敷 町 を 西 へ 下る て 、 だらだら 坂 を 降る 尽くす と 、 大きな 銀杏 が ある 。 この 銀杏 を 目標 に 右 に 切れる と 、 一 丁 ばかり 奥 に 石 の 鳥居 が ある 。 片側 は 田圃 で 、 片側 は 熊笹 ばかり の 中 を 鳥居 まで 来る て 、 それ を 潜る 抜ける と 、 暗い 杉 の 木立 に なる 。 それ から 二 十 間 ばかり 敷石 伝い に 突き当る と 、 古い 拝殿 の 階段 の 下 に 出る 。 鼠色 に 洗い出す れる た 賽銭 箱 の 上 に 、 大きな 鈴 の 紐 が ぶら下がる て 昼間 見る と 、 その 鈴 の 傍 に 八幡宮 と 云う 額 が 懸る て いる 。 八 の 字 が 、 鳩 が 二 羽 向う ある た よう だ 書体 に できる て いる の が 面白い 。 その ほか に も いろいろ の 額 が ある 。 たいてい は 家中 の もの の 射抜く た 金的 を 、 射抜く た ものの 名前 に 添える た の が 多い 。 たま に は 太刀 を 納める た の も ある 。
　 鳥居 を 潜る と 杉 の 梢 で いつ でも 梟 が 鳴く て いる 。 そうして 、 冷飯草履 の 音 が ぴちゃぴちゃ する 。 それ が 拝殿 の 前 で やむ と 、 母 は まず 鈴 を 鳴らす て おく て 、 すぐ に しゃがむ で 柏手 を 打つ 。 たいてい は この 時 梟 が 急 に 鳴く ない なる 。 それから 母 は 一心不乱 に 夫 の 無事 を 祈る 。 母 の 考え で は 、 夫 が 侍 だ ある から 、 弓矢 の 神 の 八幡 へ 、 こう やる て 是非 ない 願 を かける た 、 よもや 聴く れる ぬ 道理 は ない う と 一 図 に 思いつめる て いる 。
　 子供 は よく この 鈴 の 音 で 眼 を 覚ます て 、 四辺 を 見る と 真暗 だ もの だ から 、 急 に 背中 で 泣く 出す 事 が ある 。 その 時 母 は 口 の 内 で 何 か 祈る ながら 、 背 を 振る て あやす う と する 。 すると 旨い 泣き やむ 事 も ある 。 また ますます 烈しい 泣く 立てる 事 も ある 。 いずれ に する て も 母 は 容易 に 立つ ない 。
　 一 通り 夫 の 身の上 を 祈る て しまう と 、 今度 は 細 帯 を 解く て 、 背中 の 子 を 摺る おろす よう に 、 背中 から 前 へ 廻す て 、 両手 に 抱く ながら 拝殿 を 上る て 行く て 、 「 好い 子 だ から 、 少し の 間 、 待つ て おく で よ 」 と きっと 自分 の 頬 を 子供 の 頬 へ 擦る つける 。 そうして 細 帯 を 長い する て 、 子供 を 縛る て おく て 、 その 片端 を 拝殿 の 欄干 に 括る つける 。 それから 段々 を 下りる て 来る て 二 十 間 の 敷石 を 往 く たり 来る たり 御 百 度 を 踏む 。
　 拝殿 に 括る つける られる た 子 は 、 暗闇 の 中 で 、 細 帯 の 丈 の ゆるす 限り 、 広縁 の 上 を 這う 廻る て いる 。 そう 云う 時 は 母 にとって 、 はなはだ 楽 だ 夜 だ ある 。 けれども 縛る た 子 に ひく ひく 泣く れる と 、 母 は 気 が 気 だ ない 。 御 百 度 の 足 が 非常 に 早い なる 。 大変 息 が 切れる 。 仕方 の ない 時 は 、 中途 で 拝殿 へ 上る て 来る て 、 いろいろ すかす て おく て 、 また 御 百 度 を 踏む 直す 事 も ある 。
　 こう 云う 風 に 、 幾 晩 と ない 母 が 気 を 揉む で 、 夜の目 も 寝る ぬ に 心配 する て いる た 父 は 、 とく の 昔 に 浪士 の ため に 殺す れる て いる た の だ ある 。
　 こんな 悲 いる 話 を 、 夢 の 中 で 母 から 聞く た 。
　 庄太郎 が 女 に 攫う れる て から 七 日 目 の 晩 に ふらり と 帰る て 来る て 、 急 に 熱 が 出る て どっと 、 床 に 就く て いる と 云う て 健 さん が 知らせ に 来る た 。
　 庄太郎 は 町内 一 の 好男子 で 、 至極 善良 だ 正直 者 だ ある 。 ただ 一つ の 道楽 が ある 。 パナマ の 帽子 を 被る て 、 夕方 に なる と 水菓子 屋 の 店先 へ 腰 を かける て 、 往来 の 女 の 顔 を 眺める て いる 。 そうして しきりに 感心 する て いる 。 その ほか に は これ と 云う ほど の 特色 も ない 。
　 あまり 女 が 通る ない 時 は 、 往来 を 見る ない で 水菓子 を 見る て いる 。 水菓子 に は いろいろ ある 。 水蜜 桃 や 、 林檎 や 、 枇杷 や 、 バナナ を 綺麗 に 籠 に 盛る て 、 すぐ 見舞 物 に 持つ て 行ける よう に 二 列 に 並べる て ある 。 庄太郎 は この 籠 を 見る て は 綺麗 だ と 云う て いる 。 商売 を する だ 水菓子 屋 に 限る と 云う て いる 。 その くせ 自分 は パナマ の 帽子 を 被る て ぶらぶら 遊ぶ で いる 。
　 この 色 が いい と 云う て 、 夏蜜柑 など を 品評 する 事 も ある 。 けれども 、 かつて 銭 を 出す て 水菓子 を 買う た 事 が ない 。 ただ で は 無論 食う ない 。 色 ばかり 賞 め て いる 。
　 ある 夕方 一 人 の 女 が 、 不意 に 店先 に 立つ た 。 身分 の ある 人 と 見える て 立派 だ 服装 を する て いる 。 その 着物 の 色 が ひどい 庄太郎 の 気 に 入る た 。 その 上 庄太郎 は 大変 女 の 顔 に 感心 する て しまう た 。 そこで 大事 だ パナマ の 帽子 を 脱 って 丁寧 に 挨拶 を する た 、 女 は 籠 詰 の 一番 大きい の を 指す て 、 これ を 下さる と 云う んで 、 庄太郎 は すぐ その 籠 を 取る て 渡す た 。 すると 女 は それ を ちょっと 提げる て 見る て 、 大変 重い 事 と 云う た 。
　 庄太郎 は 元来 ｜ 閑人 の 上 に 、 すこぶる 気 作 だ 男 だ から 、 では お 宅 まで 持つ て 参る ます う と 云う て 、 女 と いっしょ に 水菓子 屋 を 出る た 。 それ ぎり 帰る て 来る ない た 。
　 いく ない 庄太郎 でも 、 あんまり 呑気 過ぎる 。 只事 じゃ 無い う と 云う て 、 親類 や 友達 が 騒ぎ出す て いる と 、 七 日 目 の 晩 に なる て 、 ふらり と 帰る て 来る た 。 そこで 大勢 寄る て たかる て 、 庄 さん どこ へ 行く て いる た ん だ いと 聞く と 、 庄太郎 は 電車 へ 乗る て 山 へ 行く た ん だ と 答える た 。
　 何 でも よほど 長い 電車 に 違い ない 。 庄太郎 の 云う ところ に よる と 、 電車 を 下りる と すぐ と 原 へ 出る た そう だ ある 。 非常 に 広い 原 で 、 どこ を 見廻す て も 青い 草 ばかり 生える て いる た 。 女 と いっしょ に 草 の 上 を 歩く て 行く と 、 急 に 絶壁 の 天辺 へ 出る た 。 その 時 女 が 庄太郎 に 、 ここ から 飛び込む で 御覧 なさる と 云う た 。 底 を 覗く て 見る と 、 切岸 は 見える が 底 は 見える ない 。 庄太郎 は また パナマ の 帽子 を 脱ぐ で 再三 辞退 する た 。 すると 女 が 、 もし 思い切る て 飛び込む ない ば 、 豚 に 舐める られる ます が 好い ござる すか と 聞く た 。 庄太郎 は 豚 と 雲右衛門 が 大 嫌 だ た 。 けれども 命 に は 易い える られる ない と 思う て 、 やっぱり 飛び込む の を 見合せる て いる た 。 ところ へ 豚 が 一 匹 鼻 を 鳴らす て 来る た 。 庄太郎 は 仕方 ない に 、 持つ て いる た 細い 檳榔樹 の 洋 杖 で 、 豚 の 鼻 頭 を 打つ た 。 豚 はぐ う と 云う ながら 、 ころり と 引っ繰り返る て 、 絶壁 の 下 へ 落ちる て 行く た 。 庄太郎 は ほっと 一 と 息 接ぐ で いる と また 一 匹 の 豚 が 大きな 鼻 を 庄太郎 に 擦る つけ に 来る た 。 庄太郎 は やむをえる ぬ また 洋 杖 を 振り上げる た 。 豚 はぐ う と 鳴く て また 真逆様 に 穴 の 底 へ 転げ込む だ 。 すると また 一 匹 あらわれる た 。 この 時 庄太郎 は ふと 気がつく て 、 向う を 見る と 、 遥 の 青 草原 の 尽きる 辺 から 幾 万 匹 か 数え切れる ぬ 豚 が 、 群 を なす て 一直線 に 、 この 絶壁 の 上 に 立つ て いる 庄太郎 を 目 懸ける て 鼻 を 鳴らす て くる 。 庄太郎 は 心から 恐縮 する た 。 けれども 仕方 が ない から 、 近寄る て くる 豚 の 鼻 頭 を 、 一つ 一つ 丁寧 に 檳榔樹 の 洋 杖 で 打つ て いる た 。 不思議 だ 事 に 洋 杖 が 鼻 へ 触り さえ する ば 豚 は ころり と 谷 の 底 へ 落ちる て 行く 。 覗く て 見る と 底 の 見える ない 絶壁 を 、 逆さ に なる た 豚 が 行列 する て 落ちる て 行く 。 自分 が この くらい 多く の 豚 を 谷 へ 落す た か と 思う と 、 庄太郎 は 我 ながら 怖い なる た 。 けれども 豚 は 続々 くる 。 黒雲 に 足 が 生える て 、 青草 を 踏み分ける よう だ 勢い で 無尽蔵 に 鼻 を 鳴らす て くる 。
　 庄太郎 は 必死 の 勇 を ふるう て 、 豚 の 鼻 頭 を 七 日 六 晩 叩く た 。 けれども 、 とうとう 精根 が 尽きる て 、 手 が 蒟蒻 の よう に 弱る て 、 しまいに 豚 に 舐める られる て しまう た 。 そうして 絶壁 の 上 へ 倒れる た 。
　 健 さん は 、 庄太郎 の 話 を ここ まで する て 、 だから あんまり 女 を 見る の は 善い ない よ と 云う た 。 自分 も もっとも だ と 思う た 。 けれども 健 さん は 庄太郎 の パナマ の 帽子 が 貰う たい と 云う て いる た 。
　 庄太郎 は 助かる まい 。 パナマ は 健 さん の もの だ う 。
